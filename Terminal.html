Lista de comandos

  • Ejecutar monitor

    sudo nodejs Raspberry-Pi-Status/server.js

  • Activar Bluethoot para el control de wii

    systemctl start bluetooth

    bluetoothctl

  • Borrar archivos

    sudo rm

  • Borrar carpetas

    sudo rm -R

  • modificar permisos de carpeta por usuario

    sudo chown usuario /var/www/html

  • modificar permisos de carpeta universales

    sudo chmod 777 /var/www/html
