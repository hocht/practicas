Nespi

Lista de comandos

  • Configuracion de raspberry

    sudo raspi-config

  • Configuracion Retropie

    sudo /home/pi/RetroPie-Setup/retropie_setup.sh

  • Ejecutar monitor

    sudo nodejs Raspberry-Pi-Status/server.js

  • Activar Bluethoot para el control de wii

    systemctl start bluetooth

    bluetoothctl

  • Borrar archivos

    sudo rm

  • Borrar carpetas

    sudo rm -R

  • modificar permisos de carpeta por usuario

    sudo chown usuario /var/www/html

  • modificar permisos de carpeta universales

    sudo chmod 777 /var/www/html

  • Ver distribucion de espacio

    ncdu /

  • Reboot

    sudo reboot

  • Shutdown

    sudo shutdown -h now

  • Change Directory

    cd /path/to/directory

  • Edit Files with Nano

    sudo nano /path/to/file.txt

  • Change owner to Pi

    sudo chown pi:pi filetobechanged

  • Change owner of folder and all files in folder to Pi

    sudo chown -R pi:pi /folder/to/be/changed

  • Make shell script executable

    sudo chmod +x yourshellscript.sh

  • list Files in Current Directory

    ls
